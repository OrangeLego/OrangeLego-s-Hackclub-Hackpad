This Hackpad is a 6 key macropad with a rotary encoder, and makes use of KMK firmware.

Features:

    3D Squirted Case (Many color choices, still don't know what I'll choose)
    EC11 Rotary encoder, as a volume knob and an audio ouput device switcher
    6 Keys, there purpose, up to you!
    KISS, Keep it simple silly(of course /s)

CAD Model:

Everything fits together using 5 M3 Bolts and heatset inserts. 4 for the case, one for the PCB.

It has 2 separate printed pieces. The base where the PCB sits and the top cover.
